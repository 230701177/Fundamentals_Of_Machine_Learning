# Fundamentals Of Machine Learning

A comprehensive repository containing Machine Learning experiments, project implementation, and observation records developed as part of the Fundamentals of Machine Learning course.

---

## Repository Contents

### 1. Project – SmartClaim
SmartClaim is a Machine Learning based project developed to demonstrate practical implementation of ML concepts including data processing, prediction models, and intelligent decision-making techniques.

#### Project Features
- Data preprocessing and cleaning
- Machine Learning model implementation
- Prediction and analysis system
- Model training and evaluation
- Dataset handling and visualization

---

### 2. Machine Learning Experiments
This repository contains 10 laboratory experiments covering important Machine Learning concepts and algorithms.

#### Included Experiments
- Experiment 1 – Introduction to ML
- Experiment 2 – Data Preprocessing
- Experiment 3 – Regression Techniques
- Experiment 4 – Classification Algorithms
- Experiment 5 – Clustering
- Experiment 6 – Decision Trees
- Experiment 7 – Naive Bayes
- Experiment 8 – Support Vector Machine
- Experiment 9 – Neural Networks
- Experiment 10 – Model Evaluation

---

### 3. Observation Note
Contains the observation record PDF including:
- Aim
- Procedure
- Code
- Output
- Result
